Firebase é uma plataforma Backend-as-a-Service (BaaS) de desenvolvimento de apps
que fornece serviços de backend hospedados, como banco de dados em tempo real,
armazenamento em nuvem, autenticação, relatório de falhas, machine learning,
configuração remota e hospedagem para seus arquivos estáticos.
