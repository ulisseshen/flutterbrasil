A versão 2.49.0 do Dart e Flutter DevTools
inclui as seguintes mudanças, entre outras melhorias gerais.
Para saber mais sobre o DevTools, confira a
[visão geral do DevTools](/tools/devtools/overview).

## Atualizações gerais

- Corrigido um bug que impedia que os botões no topo de uma tela embutida do DevTools
  fossem clicados sob certas circunstâncias. -
  [#9327](https://github.com/flutter/devtools/pull/9327)

## Histórico completo de commits

Para encontrar uma lista completa de mudanças neste lançamento, confira o
[log do git do DevTools](https://github.com/flutter/devtools/tree/v2.49.0).
