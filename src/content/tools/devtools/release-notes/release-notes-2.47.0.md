A versão 2.47.0 do Dart e Flutter DevTools
inclui as seguintes mudanças, entre outras melhorias gerais.
Para saber mais sobre o DevTools, confira a
[visão geral do DevTools](/tools/devtools/overview).

## Atualizações gerais

- Corrigido um problema onde copiar todos os logs no console exibia `null` para
  quaisquer widgets inspecionados. - [#9204](https://github.com/flutter/devtools/pull/9204)

## Histórico completo de commits

Para encontrar uma lista completa de mudanças neste lançamento, confira o
[log do git do DevTools](https://github.com/flutter/devtools/tree/v2.47.0).
